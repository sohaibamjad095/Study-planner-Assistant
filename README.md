📚 Study Planner & Assistant

A modern web-based study planner designed to help students organize subjects, manage exam schedules, and track study progress effectively.

🚀 Features

Add subjects and exam dates
Automatic priority calculation
Progress tracking dashboard
Create, complete, and delete study tasks
Subject search functionality
Study streak system 
Weekly productivity analytics
Pomodoro timer for focused study sessions
Dark mode support
Local storage data persistence
Responsive design for desktop and mobile devices
🛠️ Technologies Used

HTML5
CSS3
JavaScript
Local Storage
🎯 Project Purpose

Students often struggle to stay organized when preparing for exams and managing multiple subjects. This application provides a simple and effective way to plan study schedules, monitor progress, and stay focused on academic goals.
